Execute the four-dimensional review (CEO + Design + Engineering + DX) on the current plan or design document.

Step 1: Use the skill tool to load the gstack-plan-ceo-review skill.
Step 2: After CEO review completes, use the skill tool to load the gstack-plan-design-review skill.
Step 3: After Design review completes, use the skill tool to load the gstack-plan-eng-review skill.
Step 4: Summarize findings from all three review passes.

Note: DX (Developer Experience) review is currently not available as a standalone skill. Include DX considerations in the Engineering review pass.
